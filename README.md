# Kurmall Ecosystem Architecture

This repository contains the core shared modules, architectural patterns, and communication layers for the **Kurmall Super App**—a comprehensive, multi-service digital ecosystem designed to streamline regional trade, logistics, peer-to-peer marketplaces, and social commerce.

## 🚀 Overview
Kurmall serves as an underlying digital infrastructure that connects local merchants, service providers, and everyday users. By integrating 13 specialized modules into a single high-performance architecture, it bridges the gap between cross-border supply chains and domestic consumers.

### Key Architectural Pillars:
* **Microservices Communication:** Custom NestJS wrappers and gateway orchestrators for low-latency data fetching.
* **Highly Engaging Messenger Engine:** Custom core logic for a high-engagement, media-rich social messenger tailored for next-gen user retention.
* **Cross-Border Conversion Helpers:** Architecture snippets handling live currency fluctuations (USD/IQD) for multi-merchant digital wallets.
